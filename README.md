# Dialytica

AI-supported dialysis patient monitoring system for web and mobile platforms.

---

## Project Description

Dialytica is a healthcare-focused application designed to support dialysis patients and healthcare professionals.

The system allows:
- Nurses to record and manage patient health data
- Patients to view their own medical information
- AI to analyze patient data and provide informative feedback

---

## Features

- Patient data tracking (weight, blood pressure, dialysis duration)
- Nurse data entry system
- Patient mobile interface
- Educational content (fistula care, nutrition, medication)
- AI-powered health feedback (Gemini API)
- Real-time data synchronization between web and mobile

---

## Technologies

- Mobile: React Native (Expo)
- Backend / Database: Firebase
- AI: Gemini API
- Version Control: Git & GitHub
